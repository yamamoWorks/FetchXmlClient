Xrm FetchXml Data Provider
 .NET Framework Data Provider for Dynamics CRM FetchXml


Introduction
 This is class library to get the entity record from Dynamics CRM in the same way to get from database.
 Provide follow classes.
  - FetchXmlClientFactory
  - FetchXmlConnection
  - FetchXmlCommand
  - FetchXmlDataReader
 The features, to use the fetchxml as a substitute for sql statement in command text.
 You'll can guess how to use easily.


Features
 Command text use FetchXML instead of sql statement.
 Use "Simplified Connection to Microsoft Dynamics CRM" in Connection strings.
